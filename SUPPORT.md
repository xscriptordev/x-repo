<h2 align="center" id="before-opening-an-issue"> Before Opening an Issue </h2>
<ul>
  <li><b>For Website issues:</b> Confirm you have cleared your browser cache and are not using incompatible extensions.</li>
  <li><b>For Package issues:</b> Verify your <code>xpkg</code> and <code>xpm</code> versions are up to date and you have synced the repository database.</li>
  <li><b>Provide context:</b> Include your Node.js version (for web) or your <code>pacman.conf</code> configuration (for packages).</li>
  <li><b>Visuals:</b> Attach screenshots for UI bugs or full terminal logs for build failures.</li>
</ul>

<h2 align="center" id="response-expectations"> Response Expectations </h2>
<p align="center">
  Maintainers review issues as time allows. Reports that include clear reproduction steps for the website or <code>XBUILD</code> logs for packages are prioritized to keep the X Linux ecosystem running smoothly.
</p>
